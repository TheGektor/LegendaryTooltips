# Legendary Tooltips v1.5.6 for Minecraft 1.21.3

## What's New

- **Updated to Minecraft 1.21.3** - Full compatibility with the latest Minecraft version
- **Updated Fabric Loader** - Now requires Fabric Loader 0.16.7 or newer
- **Updated Fabric API** - Compatible with Fabric API 0.107.3+1.21.3
- **Build System Improvements** - Modernized build configuration for better stability

## Requirements

- **Minecraft**: 1.21.3
- **Fabric Loader**: 0.16.7 or newer  
- **Fabric API**: 0.107.3+1.21.3 or newer
- **Java**: 21 or newer
- **Dependencies**: Iceberg (>=1.3.0), Prism (>=1.0.11)

## Optional Compatibility

- **Roughly Enough Items (REI)**: 16.0.729 or newer
- **Just Enough Items (JEI)**: 19.5.2.66 or newer
- **Equipment Compare**: 1.3.0 or newer

## Installation

1. Make sure you have Minecraft 1.21.3 with Fabric Loader 0.16.7+
2. Download and install Fabric API 0.107.3+1.21.3 or newer
3. Download and install the required dependencies: Iceberg and Prism
4. Place the Legendary Tooltips JAR file in your mods folder
5. Launch Minecraft and enjoy enhanced tooltips!

## Changes

- Full compatibility with Minecraft 1.21.3
- Updated all dependencies to their latest compatible versions
- Improved build system stability
- Maintained all existing features and functionality
