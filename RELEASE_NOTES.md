# Legendary Tooltips v1.5.6 для Minecraft 1.21.8

## Обновления
- Обновлен до Minecraft 1.21.8
- Обновлен до Fabric Loader 0.16.0
- Обновлен до Fabric API 0.115.0+1.21.1
- Версия мода: 1.5.6

## Установка
1. Установите Fabric Loader для Minecraft 1.21.8
2. Установите Fabric API
3. Поместите файл `LegendaryTooltips-1.21.8-fabric-1.5.6.jar` в папку mods

## Совместимость
- Minecraft: 1.21.8
- Fabric Loader: >=0.16.0
- Fabric API: *
- Java: >=21

## Примечания
Это упрощенная версия мода без зависимостей от Iceberg и Prism для обеспечения совместимости с Minecraft 1.21.8.
